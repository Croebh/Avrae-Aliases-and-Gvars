Someone has to inspire the crowd, why shouldn't it be you?

This alias will allow the user to use the Zealous Presence feature of the Path of the Zealot Barbarians. It will add the effect to targets in initiative, with helpful reminder text.

**Parameters**

`-t <target>` - Chooses the targets for the bonus. Can be chained, i.e., `-t target1 -t target2`
`-i` - Ignore requirements, does not use a counter.
